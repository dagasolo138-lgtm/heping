# 生灵

《生灵》是一个运行于 GitHub Pages 的动态世界模拟游戏。世界先记录客观事实，再由传记、史书和对话系统解释这些事实。

## 开发交接规则

开发前可按需查阅 [`codex.md`](./codex.md)。**每次完成有效更新后，必须在 `codex.md` 末尾追加版本更新概述。**

## 当前阶段：人物 + 地图 + 行动 + 建造 + 居住 v0.5

### 当前能力

- 10 位村民，拥有年龄、职业、技能、性格、家庭关系、库存、事实与个人记忆。
- 起始河谷：160 米 × 120 米；1 格 = 1 米 × 1 米；河流、林地、石滩、营地和资源物件。
- 村民会取水、采集浆果、砍树、搬运资源、休息、建造与睡眠。
- A* 网格寻路；资源流为“地图 → 人物背包 → 营地库存 → 工地或消耗”。
- 集体草棚可容纳 12 人；村民会领料、送料、施工、完成建筑并入住。
- 世界时间包含黎明、白昼、黄昏、夜晚；地图有对应光照层。
- 夜晚不再派发新的生产或建造任务。住进草棚的人恢复精力更快、压力下降；无住所者在营地露宿，恢复较慢且压力上升。
- 行动与睡眠结果会写入人物 `lifeEvents`。

## 目录

```text
src/
  core/          # ID、事件总线、游戏时间、确定性随机数
  data/          # 常量和基础数据
  modules/
    people/       # 人物系统
    map/          # 地图数据、生成、查询与服务层
    settlements/  # 营地共享库存
    actions/      # 任务规划、寻路、执行、睡眠与运行时调度
    buildings/    # 建筑目录、工地、施工与居住归属
    environment/  # 昼夜阶段
  ui/map/         # 地形、光照、物件、建筑、人物令牌与交互
  styles/         # 页面样式
```

## 运行

这是纯前端 ES Module 项目。GitHub Pages 发布后直接访问 `index.html` 即可。
