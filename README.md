# 生灵

《生灵》是一个运行于 GitHub Pages 的动态世界模拟游戏。第一阶段先落地人物系统：每位居民都拥有独立、可存档的身份、状态、关系、库存和不可回写的人生事实。

## 当前阶段：人物系统 v0.1

- 10 位有姓名、年龄、性格、技能和初始物资的村民
- 人物基础状态：饥饿、口渴、精力、健康、心情、压力
- 家庭与社会关系：伴侣、手足，以及可扩展的熟悉度、好感、信任和关系标签
- 人生事实记录：为传记、史书与 AI 对话提供唯一事实来源
- 命名空间扩展槽：后续补丁可添加宗教、疾病、政治、职业等数据，不污染核心结构
- JSON 导入导出接口：后续存档系统可直接调用

## 目录

```text
src/
  core/       # ID、事件总线、游戏时间
  data/       # 常量和基础数据
  modules/
    people/   # 人物系统
  styles/     # 页面样式
```

## 运行

这是纯前端 ES Module 项目。GitHub Pages 发布后直接访问 `index.html` 即可。
