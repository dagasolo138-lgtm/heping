# 生灵

《生灵》是一个运行于 GitHub Pages 的动态世界模拟游戏。世界先记录客观事实，再由传记、史书和对话系统解释这些事实。

## 开发交接规则

开发前可按需查阅 [`codex.md`](./codex.md)。**每次完成有效更新后，必须在 `codex.md` 末尾追加版本更新概述。**

## 当前阶段：人物 + 地图 + 行动 + 建造 + 居住 + 环境 v0.6

### 当前能力

- 10 位村民，拥有年龄、职业、技能、性格、家庭关系、库存、事实与个人记忆。
- 起始河谷：160 米 × 120 米；1 格 = 1 米 × 1 米；河流、林地、石滩、营地和资源物件。
- 村民会取水、采集浆果、砍树、搬运资源、休息、建造、添柴、取暖与睡眠。
- A* 网格寻路；资源流为“地图 → 人物背包 → 营地库存 → 工地、篝火或消耗”。
- 集体草棚可容纳 12 人；村民会领料、送料、施工、完成建筑并入住。
- 世界包含黎明、白昼、黄昏、夜晚，以及晴朗、阴天、降雨、寒冷、冷雨等天气。
- 天气会影响户外移动和工作效率；降雨与寒冷会积累潮湿、受寒、压力和健康风险。
- 篝火会在夜晚、降雨和寒冷中消耗燃料，附近提供温暖；燃料低时村民会消耗营地木材添柴。
- 草棚能在内部减少雨寒暴露；人物卡会显示潮湿、受寒、住处与环境状态。
- 行动、睡眠、添柴和取暖结果会写入人物 `lifeEvents`。

## 目录

```text
src/
  core/          # ID、事件总线、游戏时间、确定性随机数
  data/          # 常量和基础数据
  modules/
    people/       # 人物系统
    map/          # 地图数据、生成、查询与服务层
    settlements/  # 营地共享库存
    actions/      # 任务规划、寻路、执行、睡眠、添柴与运行时调度
    buildings/    # 建筑目录、工地、施工与居住归属
    environment/  # 昼夜、天气、篝火、环境暴露
  ui/map/         # 地形、光照、天气、物件、建筑、人物令牌与交互
  styles/         # 页面样式
```

## 运行

这是纯前端 ES Module 项目。GitHub Pages 发布后直接访问 `index.html` 即可。
