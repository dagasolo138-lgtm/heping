# 生灵

《生灵》是一个运行于 GitHub Pages 的动态世界模拟游戏。世界先记录客观事实，再由传记、史书和对话系统解释这些事实。

## 当前阶段：人物 + 地图 + 自主行动 v0.3

### 人物系统

- 10 位有姓名、年龄、性格、技能和初始物资的村民
- 人物基础状态：饥饿、口渴、精力、健康、心情、压力
- 家庭与社会关系：伴侣、手足，以及可扩展的熟悉度、好感、信任和关系标签
- 两层记忆：`lifeEvents` 保存客观人生事实；`personal` 保存人物自己的相遇、印象与主观记忆
- 人物行动状态：当前任务、阶段、上次完成动作与累计完成次数

### 起始河谷地图

- 尺寸：160 米 × 120 米
- 精度：1 格 = 1 米 × 1 米
- 分区：16 米 × 16 米区块，共 10 × 8 块
- 地形：东侧河流、北侧林地、西侧石滩、中部营地
- 地图物件：树、石头、浆果丛、篝火、物资箱
- 人物拥有真实坐标，页面支持拖动、滚轮缩放和点击人物

### 自主行动循环

- 取水、采集浆果、砍树、搬运资源、休息
- A* 网格寻路：人物沿可通行地块移动，不能穿过河流和树木
- 行为由饥饿、口渴、精力、职业、技能、背包与营地库存共同决定
- 资源先进入人物背包，再由人物搬回营地共享库存
- 每次完成行动都会追加到人物的 `lifeEvents`，供传记与史书读取

## 目录

```text
src/
  core/          # ID、事件总线、游戏时间、确定性随机数
  data/          # 常量和基础数据
  modules/
    people/      # 人物系统
    map/         # 地图数据、生成、查询与服务层
    settlements/ # 营地共享库存
    actions/     # 任务规划、寻路、执行与运行时调度
  ui/
    map/         # 地形、物件、人物令牌与交互视图
  styles/        # 页面样式
```

## 运行

这是纯前端 ES Module 项目。GitHub Pages 发布后直接访问 `index.html` 即可。
